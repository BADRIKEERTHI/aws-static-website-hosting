# AWS Static Website Hosting

## Project Overview

This project demonstrates how to deploy a static website on AWS using Amazon S3, Amazon CloudFront, and AWS IAM.

The website files were uploaded to an Amazon S3 bucket, static website hosting was enabled, appropriate bucket permissions were configured, and the website was distributed globally using Amazon CloudFront.

This project was completed as part of the Udacity AWS Cloud Fundamentals program.
